## Features Demonstrated

Each application showcases multiple Eleva concepts:

- **Component Composition** - Breaking down UIs into reusable parts
- **Signal-based Reactivity** - State management with signals
- **LocalStorage Persistence** - Saving data across sessions
- **API Integration** - Fetching external data
- **Form Handling** - Input binding and validation
- **Conditional Rendering** - Dynamic UI based on state
- **Component Styling** - CSS for each component (not auto-scoped)
